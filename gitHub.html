How to Push and Pull Content

Below is the notes that I took on how to push and pull content in GitHub

  1. Create a directory that you intend to work in. Do this by using mkdir in Terminal
  2. Make a new repository in GitHub.
  3. Take the code in GitHub and paste it into terminal. (Make sure you are in the right CD)
  4. The previous step should add a ReadMe file to your local repository
  5. Use to command "code ." in the Terminal to open your local repository in VSCode
